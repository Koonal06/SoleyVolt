Design a modern web application UI for a digital energy platform called “SoleyVolt”.

Tagline: “Nouvo Lenerzi, Nouvo Moris.”

The platform transforms solar energy (kWh) into digital tokens and allows users to track, store, and transfer energy value.

IMPORTANT FLOW:
This should be designed as a WEBSITE experience:
1. Landing page (public)
2. User clicks "Get Started" or "Login"
3. Then enters the application dashboard interface

The system is powered by Supabase (authentication, database, real-time updates), so the UI should reflect real-time data, secure login, and user account management.

----------------------------------------

STYLE:
- Clean, modern, minimal UI
- Inspired by sustainability and solar energy
- Color palette:
  - Yellow (solar energy)
  - Green (eco / sustainability)
  - Dark blue or black (technology)
- Soft gradients, rounded cards, subtle shadows
- Professional dashboard style

----------------------------------------

1. LANDING PAGE (Single Page Website)

Top Navigation Bar:
- Logo: SoleyVolt
- Menu links:
  - Home
  - About
  - Features
  - FAQ
  - Contact Us
- Buttons:
  - Login
  - Get Started (primary CTA)

Sections (scrollable one-page layout):

Hero Section:
- Tagline: “Nouvo Lenerzi, Nouvo Moris.”
- Short description of platform
- Call-to-action button: Get Started
- Clean solar/energy illustration

About Section:
- Explain the system briefly (energy → token → wallet)
- Focus on sustainability and Mauritius

Features Section:
- Track energy usage
- Convert energy into tokens
- Transfer tokens between users
- Real-time system powered by Supabase

How It Works Section:
- Step 1: Data input (kWh)
- Step 2: Token generation
- Step 3: Wallet storage
- Step 4: Transfer and usage

FAQ Section:
- Common questions about tokens, energy, and usage

Contact Section:
- Contact form (name, email, message)
- Simple clean layout

----------------------------------------

2. LOGIN / SIGNUP SCREEN (Supabase Auth)

- Appears after clicking "Get Started" or "Login"
- Logo: SoleyVolt
- Language switch (EN / FR / CR)
- Email + password login
- Sign up option
- “Forgot password”
- Clean, secure authentication UI

----------------------------------------

3. USER DASHBOARD (Main App Interface)

- Header:
  - User name
  - Language selector
- Sidebar navigation:
  - Dashboard
  - Wallet
  - Transfer
  - Settings

Dashboard Content:
- Key stats cards:
  - Total Energy Produced (kWh)
  - Total Energy Consumed (kWh)
  - Token Balance
- Real-time indicator (live badge)
- Chart: import vs export energy
- Recent transactions list (live updates)

----------------------------------------

4. WALLET SCREEN

- Large token balance display
- Breakdown:
  - Earned tokens
  - Spent tokens
- Transaction history list
- Real-time updates

----------------------------------------

5. TRANSFER SCREEN

- Send tokens form:
  - Receiver (search user)
  - Amount
- Confirm button
- Success / error messages
- Loading state (processing)

----------------------------------------

6. ADMIN DASHBOARD

- Overview cards:
  - Total users
  - Total tokens
  - Total energy imported/exported
- Graphs:
  - System activity
  - Token flow
- User management table
- Alerts / flagged activity

----------------------------------------

7. SETTINGS / PROFILE

- User profile info
- Change password
- Enable MFA toggle
- Language selector (EN / FR / CR)

----------------------------------------

8. MULTI-LANGUAGE SUPPORT

- Language toggle visible across UI
- Support English, French, Creole
- UI adaptable for translation

----------------------------------------

- Clear hierarchy

- Responsive (desktop + mobile)
- Show loading states and real-time updates
- Smooth transitions

----------------------------------------

THEME:
- Sustainable energy + smart technology
- Feels like a national-level platform for Mauritius

OUTPUT:
Generate a complete UI system:
- Landing page (one-page with Home, About, Features, FAQ, Contact)

- Consistent design system (colors, typography, components)
